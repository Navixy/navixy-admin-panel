Для полноценной работы понадобится SASS*  http://sass-lang.com/

Развертывание проекта:
1) Необходимо скачать Sencha ExtJs 4.2.1 http://www.sencha.com/products/extjs/download/ext-js-4.2.1/2281
2) Распаковать содержимое архива в <папка проекта>/, перименовать папку ext-4.2.1.883 -> ext
3) Запусить на локально на запущеном веб-сервере
4) Переименуйте файл <папка проекта>/Config.example.js в <папка проекта>/Config.js 
4) Для конфигурации приложения отредактируйте файл <папка проекта>/Config.js
	Параметры конфигрурации:
	apiProfiles - списко доступных для приложения профилей апи
		japi - профиль для Navixy JavaAPI Server
                  apiRoot - путь до API
                  apiUrlTpl - шаблон для постоения URL до API
5) Необходимо собрать тему приложения для эетого необходимо установить
    SASS http://sass-lang.com/ и compass  http://compass-style.org/

    после этого перейти в  <папка проекта>/theme и выполнить команду  sass --compass panel_metromorph/app.scss:panel_metromorph.css --style compressed



Дополнительно  (не обязательно):

*Тема интерфейса собирается в 2 этапа:
1) Тема ExtJs http://docs.sencha.com/extjs/4.2.1/#!/guide/theming
2) Тема приложения - см. выше


* Локализация
для локализации используется сервис Transifex  https://www.transifex.com/

Требования:
 необходимо установить и инициализировать клиент http://docs.transifex.com/developer/client/ - не забудь прописать в PATH
 пример конфигурации лежит в .tx/config.example

для удобства работы с локализацией существует 2 задания для Gruntjs:

lpull - получает перевод из сервиса и применяет его к файлам локализации проекта
lpush - формирует из файлов локализации проекта JSON и отправляет его в сервис
